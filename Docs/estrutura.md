# Estrutura do Projeto

Este documento descreve a organização principal do repositório.

- `EXE/` - módulos executáveis que serão implantados em `C:\DataSmart\EXE`
- `Imagens/` - imagens de interface e marketing
- `Models/` - classes de modelo JSON e domínio
- `Services/` - serviços de atualização, download, backup, path e self-update
- `publish/` - saída de publicação para distribuição do atualizador
- `manifest.json` - manifesto de módulos GitHub
- `updater-manifest.json` - manifesto do próprio DataSmartUpdater
